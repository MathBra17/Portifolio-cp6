O porfifolio possui 3 pagina, sendo a pagina principal a apresentação, em seguida da pagina dos projetos e a pagina de contato.

Há também um formulario de busca onde só é necessário digitar.
As bibliotecas utilzadas foram tailwind para estilização, react-router para criação de rotas e react-icons
para utilizar icones no menu de buscas.

Link do projeto: https://portifoliocp6.vercel.app/
